# Study code P3-C1-001

This folder contains the code to run the code for study `P3-C1-001`. You should 
only interact with the [CodeToRun.R](https://github.com/darwin-eu-studies/P3-C1-001/blob/main/Study/CodeToRun.R)
file populating your credentials to connect to the database. Results will be 
generated in the Results folder as .csv files.

> [!IMPORTANT] 
> Please make sure that you are happy with the results before submitting to the
> DTZ.

> [!NOTE] 
> Use the shiny in the report folder to visualise the results.
